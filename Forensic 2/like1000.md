# like1000

## Descripción del Reto
Este reto de nivel intermedio (250 puntos) pertenece a la categoría de Análisis Forense (*Forensics*) y aborda el desempaquetado recursivo masivo de contenedores de archivos. El objetivo consiste en diseñar un mecanismo de automatización capaz de romper 1000 niveles de empaquetado recursivo en formato `.tar` para extraer de forma íntegra el recurso final que almacena la bandera.

## Análisis Técnico (Recursion & Tar Decompression)
Un archivo `.tar` (*Tape Archive*) funciona como un contenedor que consolida múltiples recursos y metadatos en un único bloque binario sin aplicar algoritmos de compresión nativos por defecto.

* **La Vulnerabilidad/Reto**: El autor estructuró el archivo de forma iterativa anidada (`1000.tar` contiene a `999.tar`, el cual contiene a `998.tar` secuencialmente). El procesamiento manual mediante utilidades de interfaz gráfica convencional (como WinRAR o 7-Zip) resulta operativamente inviable.
* **La Solución**: Implementación de un script de automatización en Python empleando el módulo nativo `tarfile`. El algoritmo procesa un bucle descendente indexado que extrae los elementos binarios y purga dinámicamente el contenedor del nivel anterior para optimizar el almacenamiento en el sistema de archivos local.

## Proceso de Reconstrucción
1. Se analizó el archivo inicial `1000.tar` identificando la firma y el patrón de nombres secuenciales descendentes.
2. Se desarrolló un script en Python nativo (`tar_solve.py`) estructurado con la biblioteca `tarfile` y el método `extractall()` dentro de un ciclo for regresivo desde 1000 hasta 1.
3. Se integró una rutina de limpieza automatizada mediante `os.remove()` para eliminar cada contenedor intermedio procesado y evitar el desbordamiento de basura en el directorio de trabajo.
4. Tras romper el último nivel de empaquetado en la iteración final (`1.tar`), se liberó el archivo objetivo del mapa de píxeles.
5. Se invocó el recurso recuperado empleando el renderizador integrado del entorno de desarrollo para evadir restricciones del sistema operativo, logrando el volcado visual de la bandera.
