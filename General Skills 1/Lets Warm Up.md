	# Lets Warm Up

## Descripción del Reto
If I told you a word started with 0x70 in hexadecimal, what would it start with in ASCII?

## Solución
Para resolver este reto es necesario convertir el valor hexadecimal `0x70` a texto plano (ASCII). El valor `70` en hexadecimal equivale a `112` en decimal. Al revisar la tabla de caracteres ASCII, el número 112 corresponde a la letra minúscula `p`. 

## Bandera
`picoCTF{p}`
