## Solución
Para resolver este reto, simplemente descargamos el archivo proporcionado llamado `flag`. 
El nombre del reto hace referencia al comando `cat` de Linux, el cual se utiliza para concatenar y mostrar el contenido de los archivos en la terminal. 

Si estuviéramos en una terminal de Linux, la solución sería ejecutar:
`wget [enlace_del_archivo]`
`cat flag`

Como el archivo está en texto claro, también se puede abrir directamente con cualquier editor de texto (como el Bloc de notas) para revelar la bandera oculta.
